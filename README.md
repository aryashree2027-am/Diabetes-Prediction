# Diabetes-Prediction

This is my diabetes prediction project where I have built a complete machine learning system to analyze patient health data and predict the likelihood of diabetes.
In my project, I performed data preprocessing, handled missing values and used multiple classification algorithms such as Logistic Regression, KNN, Naive Bayes, SVM, Decision Tree and Random Forest to compare performance and improve accuracy.
I also visualized the data using graphs and heatmaps for better understanding. I deployed my project using Django and worked on it entirely in VS Code, making it a full end-to-end implementation from data analysis to web deployment.
